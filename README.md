# 📊 Amazon India Sales Dashboard 2025

This project presents an interactive **Power BI dashboard** that analyzes Amazon India sales data for 2025. The dashboard provides insights into sales performance, regional trends, product categories, discounts, and delivery efficiency.

## 🎯 Project Objective
The goal of this project is to transform raw sales data into meaningful insights using data visualization and analytics techniques.

## 📌 Key Features
- Sales performance analysis
- Region-wise sales insights
- Product category comparison
- Discount impact on sales
- Delivery time analysis
- Interactive filters and slicers

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Analysis

## 📊 Dashboard Insights
The dashboard helps identify:
- Top-performing product categories
- Regional sales distribution
- Impact of discounts on revenue
- Average delivery time for orders

## 📁 Dataset
The dataset contains fields such as:
- Order Date
- Ship Date
- Delivery Date
- Region
- Product
- Category
- Quantity
- Sales
- Discount

## 🚀 Outcome
This project demonstrates how business data can be analyzed and visualized to support data-driven decision making.
